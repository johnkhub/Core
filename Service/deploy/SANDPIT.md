INSTRUCTIONS FOR STANDALONE IMPORT SYSTEM
=========================================

Software
--------
These must be installed.
* Java JRE 8
* Postgres 10
* PgAdmin 4

Configuration
-------------

### Core Service ###

FAKEAUTH=true

### Importer ###

The importer is [documented extensively in](../IMPORT.md).
> With `FAKEAUTH`, the credentials are not important as they will not be used.
